# Мой IT-проект 🚀

Привет! Это **твой** репозиторий курса «IT с нуля». Сюда ты будешь складывать всё, что сделаешь за курс — от первой странички до своего бота и анализа данных.

## Что внутри

| Папка | Для чего |
|-------|----------|
| `web/` | Веб-страницы. Здесь лежит `index.html` — твоя первая страница в интернете |
| `python/` | Программы на Python |
| `data/` | Файлы с данными (таблицы CSV и т.п.) |
| `notes/` | Твои заметки: приёмы работы с ИИ, шпаргалки, идеи |

## Как опубликовать свою страницу (GitHub Pages)

1. Нажми **Fork** вверху справа (или открой <https://github.com/eeealesha/it-osnovy-starter/fork>) — появится твоя копия этого репозитория.
2. В своей копии открой **Settings → Pages**.
3. В разделе **Build and deployment** выбери ветку `main` и папку `/ (root)`, нажми **Save**.
4. Подожди минуту и обнови страницу — GitHub покажет ссылку вида
   `https://твой-логин.github.io/it-osnovy-starter/`.
5. Открой `web/index.html` прямо на сайте GitHub (карандаш ✏️ = редактировать), впиши своё имя,
   нажми **Commit changes**. Готово — страница в сети.

> 🔒 **Важно:** всё в этом репозитории видно всем. Не публикуй пароли, телефон, адрес и другие личные данные.
